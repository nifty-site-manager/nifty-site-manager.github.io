Support

If you have found a bug or would like to request a feature with nsm then do so through the master branch of the nsm repository on GitHub.

If you have found a typo or would like to suggest a change to the nsm site then do so through the stage branch of the nsm repository on GitHub.

If you would like to ask other users for help or share what you have done yourself then I have set up an nsm subreddit on Reddit.
