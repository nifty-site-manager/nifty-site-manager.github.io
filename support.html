Support

If you have found a bug or would like to request a feature with nsm then do so through the nsm repository on GitHub.

If you have found a typo or would like to suggest a change to the nsm site then do so through the nsm site repository on GitHub.

If you would like to ask other users for help or share what you have done yourself, I have set up an nsm subreddit on Reddit.
