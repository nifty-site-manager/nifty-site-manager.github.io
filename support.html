Support

Get support and support Nift

If you have found a bug, have any questions/feedback or would like to request a feature then please do so through the issues of the official repository at GitHub. We will have an email address soon for private support, if you need this now ask through GitHub issues or on Discord.

If you have found a typo or would like to suggest a change to the Nift site then do so through one of the Nift site repositories on GitHub.

If you would like to discuss your experience with Nift or ask questions to more than just the developer, you can do so using GitHub discussions, the official Discord or official Subreddit. We also have the Twitter tag @nifty_site_man.
