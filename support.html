Support

Get support and support Nift

If you have found a bug or would like to request a feature with Nift then do so through one of the official Nift repositories on BitBucket, GitHub or GitLab.

If you have found a typo or would like to suggest a change to the Nift site then do so through one of the Nift site repositories on BitBucket, GitHub or GitLab.

If you would like to contribute financially, for example so that people might be able to afford the time it takes to develop more features or fix any bugs that arise, there is a PayPal donation link below:

If you would like to discuss your experience with Nift or ask questions to more than just the developer, you can do so using the official Discord or official Subreddit. We also have the Twitter tag @nifty_site_man.
