Installing nifty-site-manager on windows

Installing a c++ compiler

If you do not already have a c++ compiler then you will need to install one.

On Windows I typically just use code::blocks which you can get to download/install with MinGW as the c++ compiler.

Installing Git Bash and make

Install Git, which should come with Git bash as well. Get make from here, ie:
  1. Go to here;
  2. download make-version-without-guile-w32-bin.zip (get the version without guile)
  3. Extract the contents of the zip file;
  4. Copy contents to Git/mingw64.

Update your PATH variable

You may find you need to update your PATH variable, simply:
  1. Add a bash file .bashrc to ~/ containing PATH=$PATH:C:/Program\ Files\ (x86)/CodeBlocks/MinGW/bin/;
  2. Testart git bash terminal.

Installing nsm

Download nsm-master-*.zip, then follow these steps to compile and install:
  1. Extract nsm-master-* from nsm-master-*.zip;
  2. Open a git bash window (right click the desktop for example) and change directory to nsm-master-*;
  3. Compile nsm by running make;
  4. Move the compiled file nsm in to a directory contained in your PATH variable, for example I move it into "C:/Program\ Files\ (x86)/CodeBlocks/MinGW/bin";
  5. Delete both the file nsm-master-*.zip and the directory nsm-master-* containing the uncompiled code.

You should now have a functioning version of nsm installed on your Windows machine.

Uninstalling nsm

Should you ever want to remove nsm from your machine (aka commit blasphemy), simply remove the executable that you placed in a directory covered by your PATH variable.
