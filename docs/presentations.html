    Making presentations with Nift

    You can use the following frameworks with Nift to make presentations for talks:

    • reveal.js (GitHub)
    • impress.js (GitHub)
    • deck.js (GitHub)
    • Shower (GitHub)
    • inspire.js (GitHub)
    • Slippy

    For template presentation repositories that you can import (or fork and rename) then clone to start making presentations, see the table below:

    impress-template GitHub - GitLab BitBucket - GitHub - GitLab
