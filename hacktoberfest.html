Hacktoberfest

For the sixth straight year, Hacktoberfest will bring together people representing more than 150 countries, both online and in person. And while the mission stays the same, the size of the community and the number of contributions continue to grow year after year. In 2018, precisely 105,341 people signed up and contributed 401,231 pull requests to projects on GitHub. Below are projects you can get involved with using Nift as part of Hacktoberfest 2019.

Nift is a cross-platform open source website generator. It can be more than 12 times faster than Hugo at rebuilding updated pages for your website (evidence), likely making it the world's fastest framework for making websites. If you can star Nift on GitHub too that'd be awesome. The developer of Nift follows everyone who follows Nift.

Projects

Make an example website using Nift

  1. Create an example website using Nift
  2. Create a public repository with your example website
  3. Fork this repo
  4. Add a link to your repository in the examples.txt file
  5. Create a pull request

Make a template website using Nift

  1. Create a template website using Nift
  2. Create a public repository with your template website
  3. Fork this repo
  4. Add a link to your repository in the templates.txt file
  5. Create a pull request

Just to clarify

You will need to create an example/template involving Nift before you send a pull request and the link has to point to your repo in order to be approved! We will also be screening for a serious attempt to make an open source contribution, no freebie pull requests here sorry! There has been some confusion of whether it is enough to add yourself to a list. We do not support “stealing a contribution”.
