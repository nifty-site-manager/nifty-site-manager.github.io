Templates

Website repository templates for making websites

For website templates check out the following links:

  • HTML5 UP
  • Pixelarity
  • Templated
  • TemplateMo
  • TooPlate
  • Creative Tim
  • W3.CSS Templates
  • Free CSS Templates
  • Template Monster
  • OS Templates
  • Site123
  • ColorLib
  • Free Website Templates

Also check out AnyMod for all sorts of things that you can add to your websites.

If you would like any template from the above links turned in to a Nift template file an issue at GitHub and we will make it for you. It is also a useful skill to be able to take a template and create a Nift template from it, take a look at how the same has been done for templates in the tables below for how.

For template site repositories that you can import (or fork and rename) then clone to start making websites, see the table below:
