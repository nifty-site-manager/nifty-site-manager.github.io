For template SNApp repositories that you can import (or fork and rename) then clone to start making SNApps, see the table below:

name demo repository
basic-site GitHub - GitLab BitBucket - GitHub - GitLab
dev-site GitHub - GitLab BitBucket - GitHub - GitLab
evie-undraw GitHub - GitLab BitBucket - GitHub - GitLab
gradient-site GitHub - GitLab BitBucket - GitHub - GitLab
markdown-site GitHub - GitLab BitBucket - GitHub - GitLab
menus GitHub - GitLab BitBucket - GitHub - GitLab
null-site GitHub - GitLab BitBucket - GitHub - GitLab
parchment-site GitHub - GitLab BitBucket - GitHub - GitLab
simple-site GitHub - GitLab BitBucket - GitHub - GitLab
tiar-site GitHub - GitLab BitBucket - GitHub - GitLab
name demo repository

To have a template added to the above table, email templates@nift.cc with a link to a repository on BitBicket/GitHub/GitLab. If the repository indeed contains a SNApp made with Nift then we will fork/import the repository to SNApp-templates on BitBucket, GitHub and GitLab and add it to the above table.

To make changes to a template that has already been added to the above table, make a pull request from the repository you had us clone originally.
