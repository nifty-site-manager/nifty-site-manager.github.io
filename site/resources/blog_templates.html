Blog Templates

Blog repository templates for making blogs

For template blog repositories that you can import (or fork and rename) then clone to start making blogs, see the table below:

name demo repository
gradient-blog GitHub - GitLab BitBucket - GitHub - GitLab
markdown-blog GitHub - GitLab BitBucket - GitHub - GitLab
tiar-blog GitHub - GitLab BitBucket - GitHub - GitLab
name demo repository

To have a template added to the above table, email templates@nift.cc with a link to a repository on BitBicket/GitHub/GitLab. If the repository indeed contains a blog template made with Nift then we will fork/import the repository to nsm-blogs on BitBucket, GitHub and GitLab and add it to the above table.

To make changes to a template that has already been added to the above table, make a pull request from the repository you had us clone originally.

The following table contains modified HTML5 UP template blog repositories that you can import (or fork and rename) then clone to start making blogs. If there is an HTML5 UP template you would like that is not in the table below email templates@nift.cc and we will do our best to add it for you.
