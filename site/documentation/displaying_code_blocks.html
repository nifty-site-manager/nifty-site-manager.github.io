Guide to displaying code blocks

Syntax Highlighting

One easy way to have syntax highlighted code blocks on your site is to use Google Code Prettify. Using Google Code Prettify on your site is as simple as adding the following code to the head of each page: 

					<!-- uses google-code-prettify for syntax highlighting 
						 see https://github.com/google/code-prettify for more info -->
					<script src="https://cdn.jsdelivr.net/gh/google/code-prettify@master/loader/run_prettify.js">
					</script>

Note the examples on this site are using the included pre.css file as well as the script above.

See the Google code prettifier README for supported languages and other relevant information.

Also see the nsm tutorial for a better idea of how to nicely display code blocks. More to be added here at some point (probably no time soon). Alternatively, there are some examples below.
