    Syntax

    The syntax for the pull command is: 

    nift pull
nsm pull

    Description

    The pull command is for pulling changes to the local website repository from the remote repository.

    Note: While Nift uses Git to pull changes, often website repositories have multiple branches in them and Nift will pull changes to all branches with the one command, rather than having to manually pull changes from both the project root directory and the output/site/build directory.
